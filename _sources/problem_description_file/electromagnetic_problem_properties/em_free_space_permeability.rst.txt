**************************
EM Free Space Permeability
**************************

::

	EM Free Space Permeability = <float> 

-----------------------
**Description / Usage**
-----------------------

This sets the electromagnetic free space permittivity, :math:`\mu_0` for Time-Harmonic Maxwell problems

<float>                       
    Value of the electromagnetic free space permittivity

------------
**Examples**
------------

Following is a sample card:
::

	EM Free Space Permeability = {4*PI*1e-7}
