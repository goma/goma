******************************************
Post Processing Averages
******************************************

::

   Post Processing Averages =

-----------------------
**Description / Usage**
-----------------------

This card indicates that the cards between this and an *END OF AVERAGES* card are
to be read and processed. If this card (*Post Processing Averages*) is not
present, *AVERAGE* cards will be ignored.

------------
**Examples**
------------

There are no input parameters for this card, which always appears as follows:
::

   Post Processing Averages =
